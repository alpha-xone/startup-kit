# web-design-guidelines — idea-forge HTML 仪表盘设计系统

> 本文件定义 idea-forge 的默认 HTML 输出样式规范。每次运行流水线时，其 HTML 输出的视觉风格应遵循此处定义的设计系统。

---

## 一、色调板（Color Palette）

### 状态色（用于决策标签、分数、进度条）

| 状态 | 色值 | 含义 | 使用场景 |
|---|---|---|---|
| 🟢 GO | `#16a34a` (green-600) | 通过、可推进 | 综合分≥3.5、GO 标签、高分卡片 |
| 🟡 迭代 | `#d97706` (amber-600) | 待改进、需观察 | 综合分 2.5-3.5、迭代标签、中等分卡片 |
| 🔴 KILL | `#dc2626` (red-600) | 淘汰、止损 | 综合分<2.5、KILL 标签、低分卡片、止损线触发警告 |
| ⚪ 默认 | `#6b7280` (gray-500) | 中性、基准 | 未评分项、背景参考 |

### 品牌色（header / 标题 / 导航）

| 角色 | 色值 | 使用 |
|---|---|---|
| 主色 | `#1e293b` (slate-800) | 标题栏、导航背景 |
| 强调色 | `#3b82f6` (blue-500) | 链接、按钮、雷达图网格线 |
| 背景 | `#f8fafc` (slate-50) | 页面背景 |
| 卡片背景 | `#ffffff` | 评分卡片、详情区块 |
| 分割线 | `#e2e8f0` (slate-200) | 区块分隔、表格边框 |

### 雷达图配色（多 idea 叠加对比时，按此顺序循环）

## 二、排版（Typography）

### 字号层级

| 元素 | 大小 | 字重 | 备注 |
|---|---|---|---|
| 页面标题 | 1.75rem / 28px | 700 (bold) | 仪表盘顶部标题 |
| 项目名称（卡片内） | 1.125rem / 18px | 600 (semibold) | 卡片标题 |
| 综合分数（大号数字） | 2.25rem / 36px | 700 (bold) | 卡片核心数据 |
| 维度标题 | 0.875rem / 14px | 600 (semibold) | 雷达图下方、评分表头 |
| 正文 | 0.875rem / 14px | 400 (regular) | 详情描述、证据摘要 |
| 辅助文字 | 0.75rem / 12px | 400 (regular) | 时间戳、轮次号、页脚 |

### 字族

```
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "PingFang SC", "Microsoft YaHei", sans-serif;
```

## 四、图表组件（Chart Components）

### 雷达图（Spider/Radar Chart）

- **坐标系**：六轴（需求、斩杀线、时机、自身契合、分发、机会成本）
- **刻度**：0-5，每 1 分一条网格线
- **填充**：半透明填充（fill opacity: 0.15），描边线宽 2px
- **图例**：单项目时居中下方；多项目时右上角
- **实现方式**：Canvas 2D 或 SVG，优先 SVG（缩放不失真）

### 评分卡片

- **尺寸**：最小宽度 140px，内边距 16px
- **结构**：上：维度名称（小字）— 中间：分数（大号数字，状态色）— 右下：趋势箭头 ↑/↓/→（如果为 loop 复查模式）
- **hover 效果**：轻微上移（transform: translateY(-2px)）+ 阴影加深（box-shadow transition 150ms）

### 止损线列表

- **样式**：列表项 + 左侧状态图标（✅ 圆绿色 / ⚠️ 圆红底白图标）
- **触发状态**：触发线整行背景 tint red-50 (`#fef2f2`)
- **未触发**：正常背景，灰色勾

## 五、交互规范（Interaction）

- **详情折叠**：各维度详情采用 `<details>` / `<summary>` 手风琴
- **排序切换**：批量模式提供 `<select>` 下拉，选项：综合分、需求、斩杀线、分发
- **卡片 hover**：translateY(-2px) + box-shadow 加深，过渡 150ms ease
- **雷达图 tooltip**：hover 数据点显示「项目名: 维度 分数」
- **页内锚点**：每个大板块有一个 id，导航栏快速跳转

---

## 六、输出代码规范

- **单文件自包含**：所有 CSS 内联在 `<style>` 中，所有 JS 内联在 `<script>` 中，无外部依赖
- **字符集**：`<meta charset="UTF-8">`
- **视口设置**：`<meta name="viewport" content="width=device-width, initial-scale=1.0">`
- **无外部请求**：不使用 CDN 字体、图标库或框架（可选例外：Chart.js 通过 CDN 加载仅用于雷达图，但优先自实现 SVG 雷达图以保持离线可用）
- **Canvas 后备**：使用浏览器的 Canvas 2D API 绘制雷达图，不依赖第三方图表库
